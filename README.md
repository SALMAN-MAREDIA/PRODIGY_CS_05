# PRODIGY_CS_05 – Network Packet Sniffer

🌐 A Python tool to capture and display IP packets in real time.

## Features
- Captures source & destination IPs
- Identifies protocol (TCP/UDP)
- Stops sniffing on `ESC` key press

## Technologies Used
- Python
- Scapy
- Pynput
